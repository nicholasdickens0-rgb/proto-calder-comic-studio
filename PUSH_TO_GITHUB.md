Use this when the GitHub repository has been created.

```text

```

## Git Bash Commands

From Git Bash:

```bash

bash ./publish-to-github.sh
```

That script publishes to:

```text

```

## Enable GitHub Pages

1. Open the repository on GitHub.
2. Go to **Settings**.
3. Go to **Pages**.
4. Set **Source** to **Deploy from a branch**.
5. Choose branch `main`.
6. Choose folder `/root`.
7. Save.

GitHub will show the Pages URL after it finishes publishing.
